Get Your Game On.!

We are implementing a sporting goods store which can be a one stop shop for all items related to customers favourite sports. This project will facilitate the daily operations of a store by providing a database which will manage customers and also will manage the store inventory, a frontend application which will access the database and will display the result.

Note this project has been built with:

      Python DJango Framework- which is a high-level Python web framework that enables rapid development of secure and maintainable websites
      MySQL as a backend Database

To help you get started, we have also set up:

Inventory Management: Click On Inventory option present in the options tab- All store inventory is managed Here. This is basically admin panel where he can manage the inventory. Using this option, he can add a new product, delete/update existing products in inventory.

Orders: All transactions happening at the store and orders are recorded here.

LogOut:To logout from this Admin Panel
