Hello, This is a Sporting goods Store !

Get Your Game On.!

This is a sporting goods store which can be a one stop shop for all items related to your favourite sports. This project will facilitate the daily operations of a store by providing a database which will manage customers and also will manage the store inventory, a frontend application which will access the database and will display the result on UI.

Note this project has been built with:

      Python DJango Framework- which is a high-level Python web framework that enables rapid development of secure and maintainable websites
      MySQL as a backend Database
